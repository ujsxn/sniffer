================================================================================
    zhuabao 项目概述
===============================================================================

一个数据包捕获软件。

zhuabao.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，其中包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

zhuabao.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。它包含有关项目文件与筛选器之间的关联信息。在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.cpp”文件与“源文件”筛选器关联。

zhuabao.h
    这是应用程序的主头文件。
    其中包括其他项目特定的标头（包括 Resource.h），并声明 CzhuabaoApp 应用程序类。
	此外在本程序中，该应用程序文件作为数据中心。

zhuabao.cpp
    这是包含应用程序类 CzhuabaoApp 的主应用程序源文件。

zhuabao.rc
    这是程序使用的所有 Microsoft Windows 资源的列表。它包括 RES 子目录中存储的图标、位图和光标。此文件可以直接在 Microsoft Visual C++ 中进行编辑。
	自定义消息放在了zhuabao.h中。

res\zhuabao.ico
    这是用作应用程序图标的图标文件。此图标包括在主资源文件 zhuabao.rc 中。

MainFrm.h, MainFrm.cpp
    这些文件中包含框架类 CMainFrame，该类派生自
    CFrameWnd 并控制所有 SDI 框架功能。

/////////////////////////////////////////////////////////////////////////////



/////////////////////////////////////////////////////////////////////////////

其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 zhuabao.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。
	包含了Winpcap。
	包含了自定义的数据结构。

Resource.h
    这是标准头文件，可用于定义新的资源 ID。Microsoft Visual C++ 将读取并更新此文件。
